# Linux Server Review

## Description

This is a guided learning scenario. Follow this <a href="https://docs.sadservers.com/docs/scenario-guides/practical-linux-server-review/" target="_blank">Linux Server Review Scenario Guide</a>
<br><br>
The purpose of this scenario is to review a Linux server and be able to answer questions like:
<ul>
    <li>What's the purpose of the server?</li>
    <li>What's the hardware (CPU / RAM / disk / net) utilization of the server? is there a problem there?</li>
    <li>What is running and what's going on in the server?</li>
</ul>

## Test

This challenge doesn't have a specific solution (there's no "Check My Solution").
