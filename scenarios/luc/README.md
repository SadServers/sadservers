## Description

This is a Debian 11 server without a challenge; it's for you to do as you please.<br><br>
It's meant in principle for guided tutorials like the <a href="https://linuxupskillchallenge.org/" target="_blank">Linux Upskill Challenge</a>, with some limitations (there's still no outgoing Internet access).<br><br>
Once you are done with your task, you can stop (pause) your server from your <a href="/accounts/dashboard" target="_blank">dashboard</a>.<br><br>
From the dashboard at any time you can restart the stopped server to continue your tasks or you can stop the server if it's running. You can also destroy it at any time.<br><br>
After 30 mins, the server will be stopped. You can restart it from the dashboard.
<br><br>
This is a "beta" feature and there can be some issues.

For example if you stop your server with "shutdown" from inside the instance, SadServers won't be notified and the server will show up as still running while it was terminated.

## Test
